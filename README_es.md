# **GeekLab**

GeekLab es ahora más útil que nunca. Ha evolucionado de ser una simple herramienta de pruebas a convertirse en una suite completa para usuarios avanzados y desarrolladores de Android. Ya sea que necesites inspeccionar las entrañas del sistema, simular comportamientos de notificaciones o analizar la conectividad, GeekLab proporciona las herramientas necesarias en un paquete de código abierto.

## **Orígenes**

GeekLab tiene sus raíces en el foro de Xiaomi España, [Mi Community](https://c.mi.com/es/forum-2814-1.html). Originalmente diseñado para poner a prueba los límites de los dispositivos y ayudar a los usuarios a entender su hardware, el proyecto fue restaurado tras 5 años de inactividad para apoyar el desarrollo de [Hyper Bridge](https://github.com/D4vidDf/HyperBridge).

## **Funciones Principales**

GeekLab ofrece una amplia gama de funcionalidades categorizadas para una mejor accesibilidad:

### 🔔 **Herramientas de Notificación**
* **Simulador de Notificaciones**: Crea y personaliza notificaciones para probar la visualización y el agrupamiento del sistema.
* **Historial de Notificaciones**: Realiza un seguimiento de las notificaciones entrantes con un inspector integrado.
* **Simulación de Llamadas**: Prueba cómo el sistema gestiona las llamadas entrantes utilizando la API de Telecom.
* **Live Updates y Metric Styles**: Soporte para las funciones más recientes de Android como Live Updates (API 36+) y Metric Styles (API 37+).

### 📱 **Información del Dispositivo y Sistema**
* **Información de Hardware**: Resumen detallado de las especificaciones de tu dispositivo.
* **Diagnóstico de Batería**: Monitoriza la salud, el estado y los detalles técnicos de la batería.
* **Visor de Aplicaciones**: Explora las aplicaciones instaladas y sus propiedades.

### 🌐 **Conectividad y Escáneres**
* **Analizador y Escáner WiFi**: Inspecciona redes cercanas y analiza la potencia de la señal.
* **Escáner Bluetooth y BLE**: Descubre y analiza dispositivos Bluetooth clásicos y de bajo consumo (Low Energy).
* **Escáner NFC**: Lee e identifica etiquetas NFC.
* **Analizador Web**: Analiza la estructura y el rendimiento de sitios web directamente desde la aplicación.

### 📸 **Multimedia y Pantalla**
* **Explorador CameraX**: Prueba las capacidades y funciones de la cámara.
* **Soporte Ultra HDR**: Experimenta y prueba imágenes en Ultra HDR (API 34+).

### 🛠️ **Herramientas para Desarrolladores y Avanzadas**
* **Accesos Directos Rápidos**: Acceso fácil a las Opciones de Desarrollador y ajustes ocultos del sistema.
* **Herramientas Específicas para Xiaomi**: Accesos directos especiales para dispositivos Xiaomi/HyperOS, incluyendo Selector de Banda, Mejora HDR y Modo Rendimiento.
